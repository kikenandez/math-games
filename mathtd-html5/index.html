MATH TOWER DEFENSE

route each bubble through the path that drains it to zero

Bubbles enter from the left carrying a number and travel through three sequential redirect gates. At each gate click TOP / MID / BOT to switch the route (keys: Q W E for gate 1, A S D for gate 2, Z X C for gate 3). MID is short, TOP/BOT are long detours that expose bubbles to more towers. Place operator towers near the path (towers can only be placed between waves, not while a wave is in progress!) — each tower fires once per path cell it covers: a straight stretch = 1 hit, an inside corner = 2, an inside U-turn = 3. The hover preview shows the exact ×N for the active route. Reduce every bubble to 0 before it reaches the base. Survive 5 waves to level up.
