visual action training · distinguish letter reversals

Aim and fire your base turret! Helicopters fly across the sky, dropping paratroopers. At each level, you must shoot down only the target letters while letting the friendly ones land safely so they can run into the Rescue Truck!

⚠️ SABOTAGE STACKING: Landing 4 bad paratroopers on either side of the turret will blow you up! And if a bad paratrooper lands on a Rescue Truck, it explodes — game over!
